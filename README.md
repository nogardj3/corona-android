# Corona Android

Corona - Android Application

## How to use

1. local.properties에 
admob_app_id = ""
admob_banner_id = "" 
google_map_key = ""
추가

2. app/ google-services.json 파일 추가

3. strings.yaml에 서버 urls 추가
